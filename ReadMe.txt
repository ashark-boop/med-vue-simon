=== Описание проекта ===
Проект представляет из себя игру Simon The Game, 
выполненную в виде веб-приложения с помощью JavaScript фреймворка Vue.js.

=== Vue.js ===
Vue.js подключен как CDN прямо в ind.html.
Используется сборка для разработки (devtools = true по умолчанию).

=== Логика приложения ===
Вся логика приложения находится в App, ind.html. 

=== Ресурсы ===
CSS стили определены в файле styles.css.
Звуки находятся в /sounds, взяты отсюда https://github.com/kellyk/javascript-simon.
